project name  : reselhub
project purpose :ReSell Hub is a MERN stack marketplace application that connects buyers and sellers on a single platform. Buyers can purchase products securely, sellers can manage products and orders, and admins can manage users, products, and orders through a role-based dashboard. The project focuses on secure authentication, smooth order management, and an easy-to-use user experience.
live url : https://resellhub-01.vercel.app
 Key Features :

- 🔐 Secure authentication using Firebase Authentication and JWT.
- 👥 Role-based access for Admin, Seller, and Buyer.
- 🛍️ Buyers can browse products, place orders, and complete secure payments using Stripe.
- 📦 Sellers can add, update, and delete their products.
- 📋 Sellers can manage incoming orders and update order status.
- 💳 Buyers can view payment history after successful transactions.
- 👨‍💼 Admin can manage all users, products, and orders from a dedicated dashboard.
- 🚫 Admin can block or unblock buyer and seller accounts.
- 🔎 Search and filter products by category.
- 📱 Fully responsive design for mobile, tablet, and desktop devices.
- ⚡ Dynamic dashboard with real-time statistics.
- 🛡️ Protected routes and secure API endpoints using JWT.
- 🎨 Modern and user-friendly interface built with Tailwind CSS.
